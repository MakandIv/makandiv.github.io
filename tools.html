Neural Network Tools

How to work with neural networks?

A library is required to design neural networks.

Brain.js

Brain.js is a GPU accelerated library of neural networks written in JavaScript for browsers and Node.js. It is simple, fast and easy to use. It provides multiple neural network implementations as different neural nets can be trained to do different things well. All neural networks on this site were created using this library.

TensorFlow

A library from Google designed specifically for training neural networks. For processing large amounts of data, it uses a multi-level system of nodes, which expands the scope far beyond the scientific field.

Synaptic

Synaptic is a Javascript neural network library for node.js and the browser that allows you to train first and second order neural network architectures. The project contains several built-in architectures - a multilayer perceptron, a multilayer long short-term memory network, a liquid state machine (LSM), and a trainer capable of training networks.

Theano

Fundamental library with mathematical objects and neural networks. Compatible with Python 2 and 3. One of the most powerful, used for fast and high-precision calculations. Benefits: - tight integration with NumPy; - the use of GPU and CPU to increase performance; - built-in code optimization mechanisms; - there are extensions for self-testing and unit testing.

Neataptic

This library provides the ability to quickly perform neuroevolution and backpropagation for the browser and Node.js. The library contains several built-in networks - perceptron, LSTM, GRU, Nark and others.

MNIST

MNIST (Modified National Institute of Standards and Technology database) is a Arabic numeral handwriting database. There are over 60,000 handwriting samples of each digit in the database.

An example of creating one of our neural networks.

This neural network tries to recognize a handwritten digit. To train it, you will need to enter several samples of Arabic digits. To create it, you need a handwriting field. We used "canvas" as such a field. Buttons are also needed in order to record sample numbers and train the neural network. The image on the canvas is broken down into a grid of pixels. The pixelated image is saved as a vector of ones and zeros. Each such vector is written into a list, which is sent to the "brain.js" library for training the neural network. In order for the already trained neural network to analyze the input data, the "likely ()" function is called. You can download this project from this link.
