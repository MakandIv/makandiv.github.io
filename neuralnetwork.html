Neural Network

Automatic selection of the correct color

In this module, the neural network tries to match the correct color of the text to the background on which it is located. To start, click on the "Start" button.

Teaching the neural network to select the color of the text

In this module, you need to select the correct text color for each rectangle. To do this, you must click on one of them. This procedure will be repeated several times. After that, the neural network will be able to automatically select the correct text color for each background.

Automatic digit recognition

In this module you can interact with a neural network. You are drawing an Arabic digit, and the neural network is trying to understand what kind of figure.

Teaching to recognize digits

In this module, you can train a neural network to recognize Arabic digits. To do this, you need to enter several sample numbers. The neural network will learn from them and will be able to recognize it independently.
