How to work with neural networks?

To work effectively with neural networks, you can use the following tools.

TensorFlow

A library from Google designed specifically for training neural networks. For processing large amounts of data, it uses a multi-level system of nodes, which expands the scope far beyond the scientific field.

TensorFlow is a comprehensive open source machine learning platform. It has a comprehensive, flexible ecosystem of tools, libraries, and community resources that enables researchers to drive the latest advances in machine learning, and it is easy for developers to build and deploy machine learning-based applications.

Synaptic

Synaptic is a Javascript neural network library for node.js and the browser that allows you to train first and second order neural network architectures.

This library includes a few built-in architectures like multilayer perceptrons, multilayer long-short term memory networks (LSTM), liquid state machines or Hopfield networks, and a trainer capable of training any given network, which includes built-in training tasks/tests like solving an XOR, completing a Distracted Sequence Recall task or an Embedded Reber Grammar test, so you can easily test and compare the performance of different architectures.

Theano

Theano has been developed since 2007 mainly by the MILA group at the University of Montreal and is named after the ancient Greek woman philosopher and mathematician Feano (presumably pictured).

Fundamental library with mathematical objects and neural networks. Compatible with Python 2 and 3. One of the most powerful, used for fast and high-precision calculations. Benefits: - tight integration with NumPy; - the use of GPU and CPU to increase performance; - built-in code optimization mechanisms; - there are extensions for self-testing and unit testing.

Neataptic

This library provides the ability to quickly perform neuroevolution and backpropagation for the browser and Node.js. The library contains several built-in networks - perceptron, LSTM, GRU, Nark and others.

MNIST

MNIST (Modified National Institute of Standards and Technology database) is a Arabic numeral handwriting database. There are over 60,000 handwriting samples of each digit in the database.

The data consists of pre-prepared sample images, on the basis of which the systems are trained and tested. The database was created after revising the original 20x20 pixel set of NIST black and white samples.

There have been numerous attempts to achieve minimal error after training on the MNIST database, which have been discussed in the scientific literature. Record results were indicated in publications devoted to the use of convolutional neural networks, the error rate was brought to 0.23%. The creators of the database themselves have provided several testing methods. The original work indicates that the use of support vector machines achieves an error rate of 0.8%.

Some works note the high results of systems built on ensembles of several neural networks; at the same time, the quality of digit recognition for the MNIST base turns out to be comparable to that of a person, and for other recognition tasks, in particular, for road signs, it is even twice as good.

This neural network tries to recognize a handwritten digit. To train it, you will need to enter several samples of Arabic digits. To create it, you need a handwriting field. We used "canvas" as such a field. Buttons are also needed in order to record sample numbers and train the neural network. The image on the canvas is broken down into a grid of pixels. The pixelated image is saved as a vector of ones and zeros. Each such vector is written into a list, which is sent to the "brain.js" library for training the neural network.
