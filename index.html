Machine Learning

About Us

This project was made for the week of student science at UlSTU. This project is dedicated to machine learning. The idea of this project came to us when we were on distance learning. Our English teacher asked us a lot of written work, when we did it, we found out that he demanded that we send him not photographs of handwritten text, but a text document with printed text. We didn't want to rewrite everything by hand, so we found the "ScreenOCR" application to translate handwritten text into printed text. This application works on the basis of machine learning, so we are interested in the possibilities of this technology. For this project, we wrote about the history of machine learning, about modern discoveries in this area, about tools that allow you to work with machine learning, and also, for example, implemented several neural networks.
